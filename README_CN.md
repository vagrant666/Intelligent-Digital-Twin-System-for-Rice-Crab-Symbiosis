# 稻蟹共生智能数字孪生系统
工业级全栈农业智能决策开源平台

## 项目简介
本系统面向温带平原稻蟹种养农场构建数字孪生全流程平台，**全程仅使用NASA、FAO官方公开真实时序数据，无任何仿真模拟数据**。
集成多套前沿深度学习模型、多目标优化算法、无人机路径规划、时序风险评估能力。
标准化PyTorch训练流水线，支持1000轮完整真实迭代、早停防过拟合、学习率衰减、断点续训。

## 数据源说明
1. NASA POWER：连续10年逐日气象观测数据（温度、湿度、辐射、降雨、蒸发、风速）
2. FAOSTAT：全球水稻、淡水甲壳类产量、产值、农资成本官方统计
3. 农业公开数据集：土壤氮磷钾、pH、病虫害风险时序数据
4. 遥感NDVI数据集：水稻作物生长植被特征时序数据

## 算法体系
### 深度学习模型
- CNN-Transformer混合时序特征提取模型
- LSTM-Attention注意力时序预测模型
- 残差回归产量预测网络
- SSA麻雀算法优化BP神经网络

### 智能优化算法
- NSGA-II四目标种养方案优化
- GA遗传算法农田无人机巡检路径规划
- ARIMA时序产量趋势预测
- 随机森林集成综合评估

## 训练机制
- 最大迭代轮次：1000 epoch
- 数据集划分比例：训练70% / 验证15% / 测试15%
- 指数学习率动态衰减
- 早停策略（30轮无下降终止训练）
- 自动保存损失、评价指标可视化曲线
- 模型权重断点续训
