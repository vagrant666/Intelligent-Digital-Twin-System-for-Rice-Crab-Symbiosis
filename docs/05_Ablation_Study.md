# 消融实验分析报告
## 一、实验目的
逐项移除模块/特征/训练策略，量化各组件精度增益，验证架构无冗余。

## 二、统一控制条件
固定数据集、迭代轮次、学习率，基准模型CNN-Transformer。

## 三、模型模块消融
1. 移除Transformer仅保留CNN：R²下降0.081，证明长时序依赖是核心增益。
2. LSTM移除注意力：关键生长期权重稀释，中长期误差大幅上升。
3. 取消高斯噪声增强：测试R²下降0.042，模型轻微过拟合。
4. 关闭学习率衰减：训练后期损失剧烈震荡，无法收敛最优值。

## 四、输入特征消融
1. 删除全部气象特征：R²暴跌至0.71，气候是产量决定性特征。
2. 删除土壤+NDVI长势：无法区分土地肥力差异，预测误差激增。

## 五、优化算法消融
NSGA改为单目标最大化产量：产量小幅提升，但成本、病虫害风险大幅上涨，无落地价值；证明多目标平衡是农业刚需。

## 六、消融总结
项目全部模块、特征、训练策略均存在正向精度增益，架构设计完整无多余组件。
