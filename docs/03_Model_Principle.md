# 模型与算法原理文档
## 一、深度学习时序预测模型
1. CNN-Transformer混合模型
一维CNN提取短期局部生长特征，Transformer编码器捕捉长周期时序依赖；兼顾短期波动与年度周期，系统默认主模型，输出水稻、河蟹双产量。

2. LSTM-Attention模型
LSTM循环提取递进时序特征，注意力机制自动放大分蘖、灌浆关键生长期权重，弱化淡季噪声，中长期预测效果优秀。

3. 残差回归网络ResNet
多层残差结构消除深层梯度消失，展平12步时序特征，拟合种养参数与产量复杂非线性耦合关系。

4. SSA-BP麻雀优化神经网络
基础BP易陷入局部最优；麻雀搜索算法全局优化网络初始权重，小样本下泛化能力更强，轻量化推理速度快。

## 二、智能优化算法
1. NSGA-II多目标优化
变量：水稻种植密度、河蟹投放密度、月度施肥量
四冲突目标：最大化水稻产量、最大化河蟹产量、最小化农资成本、最小化种养风险
输出帕累托最优解集，提供多套均衡方案供人工决策。

2. GA遗传无人机路径规划
农田巡检点位转为TSP旅行商问题；选择、顺序交叉、交换变异迭代，求解全覆盖最短巡检航线。

3. ARIMA时序预测
ADF平稳检验自动差分，网格搜索最优p/q阶，预测未来30天产量并输出95%置信区间。

4. 四维加权风险评估
权重：病虫害0.35、气候0.3、成本0.2、长势0.15；加权计算0-100综合风险，分为低/中/高三等级。
