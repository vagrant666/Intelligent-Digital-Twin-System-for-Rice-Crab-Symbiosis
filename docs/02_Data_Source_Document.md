# 数据源说明文档
## 一、数据源总规则
系统仅实时联网拉取2015-2025官方API数据，不内置离线数据集；网络异常直接抛出报错，无自动生成仿真数据逻辑。

## 二、五大数据源详情
1. NASA POWER气象API
地址：https://power.larc.nasa.gov/
指标：日均温、湿度、风速、太阳辐射、降雨量、蒸散发；粒度逐日，经纬度39.31/117.33
用途：气候特征输入、气候风险评估、产量预测核心特征

2. FAOSTAT农业统计API
地址：https://faostatservices.fao.org/api/v1/en/data/Production
指标：水稻/河蟹年产量、产值、肥料饲料成本；区域亚洲，跨度10年
用途：经济效益建模、成本风险打分、产量基准对比

3. 土壤养分开放接口
指标：氮、磷、钾、pH时序采样数据；温带稻蟹农田分区
用途：土壤肥力特征输入，作物生长适配判断

4. 病虫害&农产品市场接口
指标：月度病虫害发生率、稻蟹市场价格波动
用途：病虫害风险、经营成本风险量化

5. NDVI遥感植被接口
指标：归一化植被指数，按月匹配气象日期
用途：作物长势监测、生长风险计算

## 三、统一数据处理规范
1. 时间主键多表左连接融合
2. 缺失值线性插值填充
3. 3倍标准差剔除极端异常样本
4. MinMax归一化至[0,1]区间
5. 严格时序划分：训练70%/验证15%/测试15%，禁止随机打乱

## 四、数据目录划分
data/raw：原始API CSV文件
data/processed：清洗融合、归一化数据集+缩放器
data/dataset：时序窗口npy张量数据集
data/checkpoint：训练模型权重文件
