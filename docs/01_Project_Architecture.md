# 项目架构文档 - Rice-Crab Digital Twin System

## 一、项目概述
本项目为稻蟹共生智能数字孪生系统，面向现代化生态农业全栈智能决策平台。全部数据源来自NASA、FAO官方联网API，无仿真模拟数据，一站式完成数据采集、AI建模、多目标优化、可视化分析。

## 二、五层分层架构
1. 数据采集层
对接NASA气象、FAO农业统计、土壤养分、病虫害市场、NDVI遥感五大官方数据源，统一封装请求、缓存、重试机制，保障数据稳定获取。

2. 数据处理层
标准化流水线：去重→时间对齐→缺失插值→3σ异常过滤→多表时序融合→MinMax归一化→时序窗口数据集划分，严格保持时序顺序不打乱。

3. 模型算法层
### 深度学习预测模型
CNN-Transformer、LSTM-Attention、残差回归、SSA-BP神经网络，用于稻蟹双产量同步预测。
### 智能优化算法
1. NSGA-II四目标优化：最大化水稻、河蟹产量，最小化种养成本与种植风险
2. GA遗传算法：农田全覆盖无人机巡检TSP路径最优求解
3. ARIMA时序模型：产量中长期趋势预测
4. 加权打分风险评估：病虫害、气候、成本、长势四维风险分级

4. 训练推理层
基于PyTorch标准化训练管线，动态学习率衰减、早停防过拟合、断点续训，RMSE/MAE/R²三大评价指标，自动输出训练日志与曲线。

5. 可视化输出层
自动生成损失、精度、帕累托、巡检路径图表；一键导出多工作表Excel综合报告。

## 三、技术栈清单
1. 编程语言：Python3.10
2. 深度学习：PyTorch 2.4.1
3. 数据处理：Numpy、Pandas、Scikit-learn
4. 优化库：pymoo、自研SSA/GA
5. 时序分析：statsmodels
6. 绘图：Matplotlib
7. 部署：Windows批处理、Linux Shell一键脚本

## 四、核心工程优势
1. 纯官方真实时序数据，无离线模拟兜底
2. 模块化解耦，可单独调用任意子模块
3. 全自动化流水线，一键从采集到出报告
4. 工业级训练策略，缓解过拟合、支持断点续训
5. 覆盖产量预测、种养优化、无人机巡检、风险预警全农业场景
